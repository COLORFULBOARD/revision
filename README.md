## Getting Started

### Installation

To install the package, simply run:

```sh
$ pip install revision
```

Or install from source:

```sh
$ make install
```

### Running Lint

```sh
$ make lint
```

### Running Tests

```sh
$ make test
```

## Usage

Revision expects a configuration file to be present in the project root.

```sh
$ revision init
```
